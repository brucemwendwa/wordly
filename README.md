# Wordly Dictionary SPA

Wordly is a single-page dictionary application that allows users to search for words and view their meanings instantly without refreshing the page.

## Features
- Search for any English word
- View definitions, pronunciation, and examples
- Audio pronunciation playback
- Save favorite words
- Search history (recent searches)
- Dark mode toggle
- Offline fallback (loads last searched word)
- Responsive and minimal UI

## Technologies Used
- HTML
- CSS
- JavaScript (Vanilla)
- Free Dictionary API

## How to Run
1. Clone the repository
2. Open `index.html` in your browser
3. Start searching words
